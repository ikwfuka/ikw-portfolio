「HIBI HOME」ハウスメーカーのコーポレートサイト(架空サイト)

北海道にて注文住宅の企画・設計・施工を行うハウスメーカーのコーポレートサイトを制作しました。（架空サイト）

URL

https://x.gd/phOjH

担当

デザイン・コーディング

制作期間

ワイヤーフレーム:14時間　デザイン:20時間　コーディング:50時間

使用ツール

XD・photoshop・illustrator・Visual Studio Code

サイトの目的

企業の認知と相談・お問い合わせ件数の獲得

ターゲット

コロナにより多様な働き方ができるようになり、好きな場所で理想的な暮らしをしたいと思っている方。
北海道の自然に魅力を感じている30代～40代のファミリー層。

コンセプト

自然エネルギーを最大限に生かし、自然と共存しながら日々の暮らしができるたったひとつの家づくり

情報設計

共感から相談・お問い合わせに繋げる導線を考えて設計しました。

デザインについて

自然の中に居るような心地よさを表現するため余白にこだわりました。 自然をイメージしてメインカラーを緑、太陽や光をイメージしアクセントカラーにオレンジ、大地をイメージしてベースカラーにはベージュを使用しています。
サイトを訪れた人の印象に残るように、footer部分の背景に家の形をした画像を配置しました。

コーディングについて

統一感が出るよう、見出しや各セクションのパディングやマージンを意識して制作しました。 ヘッダー部分にコンバージョンボタンをおいていますが、少し大きめにしてクリックのしやすさを重視しました。
SassでのSCSS記法でコードを記述し、変数を使用するなどして効率良く記述できるよう心がけました。
