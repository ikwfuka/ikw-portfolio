じゃんけんゲーム（架空）

JavaScriptでコンピューターと対戦が出来るじゃんけんゲームを制作しました。

URL

https://x.gd/4Thfv

制作期間

10時間

使用ツール

Visual Studio Code

言語

HTML・JavaScript

ゲームの仕様・ルール

・最大で20試合し、得点の高い方か、先に10点取った方がゲームの勝者です。
・画面上部に、対戦回数(現在回/20回)、プレイヤー（あなた）の勝った回数、コンピューターの勝った回数を表示します。
・「じゃんけん...」と表示し、プレイヤー（ユーザー）からの「グー・チョキ・パー」の入力待ちをします。
・プレイヤーがグーチョキパーを選択したら、コンピューターはランダム手をとり、じゃんけんの結果を表示します。
・「再チャレンジ」ボタンを押すと次の試合に移ります。
・じゃんけんの結果を表示する時に、ゲームオーバーの判定をします。
(ゲームオーバーの判定：20試合を超えた、プレイヤーかコンピューターが先に10点取った)
・ゲームオーバーの時はゲーム終了画面を表示します。
「もう一度プレイする」ボタンを押すとゲームを初期化し、最初の状態にもどって新しいプレイを開始できます。

コーディングについて

DOM操作、イベント処理、条件分岐等を使用し組み立てました。
最後にリファクタリングをし、分かりやすく見通しの良いコードになるよう心掛けています。
